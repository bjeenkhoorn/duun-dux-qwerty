Duun language Unicode keyboard to type all special characters on a QWERTY keyboard.

Langue Duun Clavier Unicode pour QWERTY incl caractères speciaux.

This keyboard outputs mostly precomposed (NFC) Unicode characters. However, please be aware that it is not completely NFC. Also, not all of the NFC characters will be on the mobile longpress keys. If they are not available as precomposed characters, you may have to type the vowel followed by the diacritic (which is on a special diacritic key next to the spacebar).
